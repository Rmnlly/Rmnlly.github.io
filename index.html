A simple application that makes use of the TMDb API which asks users for their birth year and then queries the api for the movie that grossed the most that year. It then displays the movie poster and name. The Bulma css framework was also used for this.

Conway's Game Of Life

A life simulation based on John Horton Conway's original simulation. It creates and removes cells based on a set of rules that are meant to simulate a very simple version of life. Zim.js was used for the display of the cells on a canvas.
